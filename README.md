# BlazeDemo Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Features
- Page Object Model
- Data Driven Testing
- Smoke and Functional Test Groups
- Screenshot Capture
- TestNG XML Execution

## Test Cases
- Homepage validation
- Flight search
- Flight booking
- Multiple booking datasets

## Run Tests

Right Click testng.xml
→ Run As
→ TestNG Suite
